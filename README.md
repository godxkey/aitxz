# aitxz
一个用原生js编写的智能推箱子
语言:javascript  
渲染:canvas  
##项目文件目录说明  
img 放项目需要的图片文件  
js 放项目需要的脚本文件  
instance 放控制游戏进程的js文件  
render 放控制渲染对象的js文件  
robot.js 是机器人对象的js文件  
后面会详细对游戏的各个流程进行详细说明
